# RPG Game

This is a fighting game where you can choose your hero and beat every enemy that stand on his way! Fight Fight Fight!

## Feature List

 * Choose a hero
 * Choose an enemy
 * Attack with a sword
 * Attack with a magic
 * Shield
 * Display health bar
 * SFX & BGM
